How to Play

  • Guess the secret 5-digit code (digits 0–9, repeats allowed).
  • You have 6 attempts. Click the number squares to build your guess, then press ENTER.
  • After each guess, each square is colored to show how close you were:
Correct digit, correct position
Correct digit, wrong position
Digit not in the code
